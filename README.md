PROYECTO SPRING BOOT - Gestión de perfiles

Descripción:
El proyecto consiste en una aplicación java que aplica distintos perfiles para configurar entornos de producción.

Para ello, se utilizaron las siguientes tecnologías:

-Java 21
-Spring Boot 3.5.7
-Maven
-Intellij IDEA como IDE
-Profiles de Spring Boot: application-dev.properties y application-prod.properties

Instrucciones:
1. Clonar el repositorio:
https://github.com/luchi118/TP-Fundamentos-de-SPRING-BOOT.git

2. Ejecutar el proyecto, eligiendo el perfil en el archivo application.properties:
spring.profiles.active=dev (para el perfil dev)
spring.profiles.active=prod (para el perfil prod)
